# Task 2 - EDA & Business Intelligence Project

## Objective
The goal of this project is to perform Exploratory Data Analysis (EDA) and generate business insights using Python.

## Dataset
Sales dataset containing customer orders, product categories, sales amount, customer details, and city information.

## Steps Performed
- Loaded dataset using pandas
- Performed descriptive statistics
- Analyzed missing and duplicate values
- Created sales distribution histogram
- Performed category analysis using bar charts
- Generated business insights from sales data
- Exported EDA output dataset

## Files in this Project
- Apexplanet_Task2.ipynb → Main notebook with all code
- EDA_Output.csv → Final processed dataset
- ApexPlanet_DataAnalytics_Dataset.xlsx → Raw dataset
- sales_distribution.png → Sales distribution graph
- category_count.png → Category count graph
